About

App and model information

This app predicts credit risk (default vs non-default) for loan applicants using a trained CatBoost model. You provide applicant and loan details; the model returns a binary outcome used for decision support.

Features used include age, income, home ownership, employment length, loan intent, loan grade, loan amount, interest rate, percent of income, prior default flag, and credit history length. The model and encodings are loaded from the models/ directory.

Built with Flask and Tailwind CSS. For setup and run instructions, see the project README.
